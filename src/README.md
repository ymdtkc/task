# タスク管理ツール

Reactベースの高機能タスク管理アプリケーションです。アイゼンハワーマトリクスを使ったタスクの整理と、直感的なドラッグ&ドロップ操作が特徴です。

## 機能

### 📋 基本機能
- **タスクの追加・編集・削除**: 直感的なフォームでタスクを管理
- **重要度・緊急度の設定**: 3段階での分類
- **完了時間の設定**: 「すぐ終わる」「時間かかる」の2段階
- **今日のタスク管理**: 特定のタスクを今日やることとして管理

### 🗂️ 表示モード
1. **一覧表示**: 全タスクをリスト形式で表示
2. **今日表示**: 今日やるタスクのみを表示
3. **マトリクス表示**: アイゼンハワーマトリクス（9象限）での整理

### 🎯 アイゼンハワーマトリクス
- **ドラッグ&ドロップ**: タスクを象限間で移動可能
- **9象限表示**: 重要度×緊急度の組み合わせで表示
- **視覚的分類**: 色分けされた象限で優先度を把握

### 🔄 並び順機能
タスクは以下の優先順位で自動ソート：
1. 重要度（高 → 低）
2. 緊急度（高 → 低）
3. 完了時間（すぐ終わる → 時間かかる）
4. 作成日時（新しい → 古い）

### 💾 データ永続化
- ローカルストレージを使用してブラウザに保存
- リロード後もデータが保持される

## 技術スタック

- **React 18** - UIライブラリ
- **TypeScript** - 型安全性
- **Tailwind CSS v4** - スタイリング
- **shadcn/ui** - UIコンポーネント
- **react-dnd** - ドラッグ&ドロップ機能
- **Vite** - ビルドツール
- **Lucide React** - アイコン
- **Sonner** - トースト通知

## セットアップ

### 必要な環境
- Node.js 18以上
- npm または yarn

### インストール手順

1. プロジェクトをクローンまたはダウンロード
```bash
git clone <repository-url>
cd task-management-app
```

2. 依存関係をインストール
```bash
npm install
```

3. 開発サーバーを起動
```bash
npm run dev
```

4. ブラウザで `http://localhost:5173` を開く

### その他のコマンド

```bash
# プロダクションビルド
npm run build

# ビルド結果をプレビュー
npm run preview

# Lintチェック
npm run lint
```

## 使い方

### 1. タスクの追加
- 一覧タブではフォームが常時表示
- 今日・マトリクスタブでは「新しいタスク」ボタンから追加
- タイトル、説明、重要度、緊急度、完了時間を設定

### 2. タスクの管理
- チェックボックスでタスクの完了/未完了を切り替え
- 編集ボタンでタスクの内容を変更
- 削除ボタンでタスクを削除
- 「今日に追加」ボタンで今日やるタスクとして設定

### 3. マトリクス表示の活用
- タスクをドラッグして他の象限に移動
- 色分けされた象限で優先度を視覚的に把握
- 各象限には説明テキストが表示

### 4. 効率的な使い方
1. まず一覧でタスクを大量追加
2. マトリクスで重要度・緊急度を調整
3. 今日タブで当日のタスクに集中
